# Instructions du projet

## Contexte
Ce projet contient des applications web simples créées pour apprendre le vibe coding. Chaque application est un fichier HTML unique auto-suffisant.

## Règles techniques
- Toujours produire UN SEUL fichier HTML auto-suffisant
- Utiliser Tailwind CSS via CDN pour le design
- JavaScript inline (pas de fichiers séparés)
- Pas de framework lourd (pas de React, pas de Vue sauf si explicitement demandé)
- Noms de fichiers en français avec des tirets : calculateur-roi.html, generateur-devis.html

## Règles de design
- Palette par défaut : Bleu #246EAE en accent, gris anthracite #1F2A44 pour le texte, fond blanc #ffffffff
- Typographie : Lato pour les titres et le corps de texte. Style moderne, épuré et sans empattement.
- Design aéré, professionnel, style Stripe/Linear
- Mobile-friendly par défaut

## Règles de communication
- Répondre en français
- Expliquer ce que tu fais en langage simple, pas de jargon
- Demander confirmation avant de supprimer ou écraser un fichier existant
